de données extrait de BASIAS (Base de données des Sites Industriels et Activités de Services) dans le périmètre du bassin-versant du littoral marseillais à partir du site : www.georisques.gouv.fr.
La base attributaire a été retravaillée. Les champs décrivant les sites ont été conservés :
- REFERENCE : identifiant BASIAS du site
- COMMUNE : commune de localisation du site
- ETAT_SITE : état d'exploitation du site
- DEB_ACT : date de début d'exploitation du site ( format : jjmmaaaa)
- FIN_ACT : date de fin d'exploitation du site (format : jjmmaaaa)
- CODE_ACT : code des types d'activités exercés par le site selon la NAF, chaque code est séparé par une ","

D'autres champs issus de traitements ont été ajoutés :
- DEB_DEC : Reprend l'information du champ "DEB_ACT" agrégé à la décennie (ex : 11091981 => 1980)
- NB_ACT : nombre d'activités successives du site à partir du champ "CODE_ACT"
- act_unique_NV1 : définit si le site a ou non exercé un seul type d'activité d'après le niveau 1 de la nomenclature NAF
- Act1 à 6 : agrégation des types d'activités au niveau 1 de la NAF et séparation des activités successives par colonne
- NOM_BV : nom du sous bassin-versant sur lequel est localisé le site

Dans le cadre du projet BEEST, ce tableau peut être spatialisé par jointure attributaire basé sur le champ REFERENCE avec la couche SIG "BASIAS_BEEST.shp" .
